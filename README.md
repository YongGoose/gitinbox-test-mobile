# gitinbox-test-mobile

Dogfooding sandbox for **[GitInbox](https://github.com/YongGoose/GitInbox)**.

Swift Package Manager project pinned to **known-outdated** lower-bounds so
Dependabot (swift ecosystem) produces real bot PRs that feed GitInbox's
Bots lane.

## Bots

| Bot | Config | Status |
| --- | --- | --- |
| Dependabot (swift + github-actions) | `.github/dependabot.yml` | on |
| CodeRabbit (free) | GitHub App | install later |

Part of the GitInbox M0 pre-flight milestone — see
[`docs/06`](https://github.com/YongGoose/GitInbox/blob/main/docs/06-implementation-plan.md)
and [`docs/11 §3.8`](https://github.com/YongGoose/GitInbox/blob/main/docs/11-session-log.md).
